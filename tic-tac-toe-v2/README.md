# 🎮 Tic Tac Toe (GUI + OOP)

A fully functional Tic Tac Toe game built with Python using Object-Oriented Programming (OOP) and Tkinter GUI.

---

## 🚀 Features

- 🎯 Player vs Player mode
- 🤖 Player vs Bot mode
- 🧠 Smart game logic with win/draw detection
- 🖱️ Interactive GUI built with Tkinter
- 🔄 Restart game functionality
- 💡 Clean OOP architecture

---

## 🧱 Project Structure

```
tic-tac-toe/
├── main.py       # Entry point
├── gui.py        # Tkinter UI and screen management
├── game.py       # Core game logic (win/draw detection)
├── player.py     # Player class
└── bot.py        # Bot logic
```

---

## ▶️ How to Run

1. Clone the repository:
```bash
git clone https://github.com/vidushanpathir505-bot/tactic-toe-python.git
```

2. Navigate to the project folder:
```bash
cd tic-tac-toe-v2
```

3. Run the game:
```bash
python main.py
```
